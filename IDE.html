VS code

  1. Description Gérérale

    • Visual Studio Code (VS Code) est un éditeur de code open-source, gratuit et multi-plateforme, développé par Microsoft. VS Code est développé avec Electron et exploite des fonctionnalités d'édition avancées du projet Monaco Editor. Principalement conçu pour le développement d'application avec JavaScript, TypeScript et Node.js.

  2. Principale fonctionnalités

    • Plate-formes suportées: Windows, MacOs, Linux
    • Outils de collaboration : LiveShare (Extension), intégration native avec Git
    • Complétion intelligente et mise en évidence de la syntaxe: IntelliSence
    • Débogage intégré
    • Ligne de commande intégrée

  3. Forces

    • Peut être utilisé pour développer dans à peu près tous les langages de programmation
    • Disponibilité d'un très grand nombre d'extensions
    • Disponible pour les plate-forme Windows, MacOs et Linux
    • Gratuit


  4. Limites

    • C'est plutôt un éditeur très performant qu'un vrai IDE
    • Il est principalement orienté autour des fichiers, pas des projets.
    • Il ne prend pas en charge le système de contrôle de version de Microsoft

Visual Studio

  1. Description Gérérale

    • Visual Studio est un ensemble complet d'outils de développement permettant de générer des applications web ASP.NET, des services web XML, des applications bureautiques et des applications mobiles.

  2. Principale fonctionnalités

    • Plate-formes suportées: Windows, MacOs
    • Outils de collaboration intégrée, intégration native avec Git
    • Complétion intelligente et mise en évidence de la syntaxe
    • Débogage intégré
    • Ligne de commande intégrée

  3. Forces

    • C'est un IDE complet qui permet de développer des applications conformes aux spécifications .NET


  4. Limites

    • Il ne prend pas en charge le langage de programmation Java
    • Les outils de débogage sont plus ou moins complets selon les versions Communauté, Professionelle ou Entreprise

IntelliJ IDEA

  1. Description Gérérale

    • IntelliJ IDEA également appelé « IntelliJ », est un environnement de développement intégré (IDE) destiné au développement de logiciels informatiques reposant sur la technologie Java.

  2. Principale fonctionnalités

    • Plate-formes suportées: Windows, MacOs, Linux, FreeBSD, OpenVSD, Solaris
    • Outils de collaboration Git, CSV, Azure DevOps, Mercurial, Subversion
    • Complétion intelligente et mise en évidence de la syntaxe
    • Débogage intégré

  3. Forces

    • C'est un IDE complet pour le développement d'application Java


  4. Limites

    • Limité quant au nombres de langages pris en charge

Eclipse

  1. Description Gérérale

    • Eclipse est un projet, décliné et organisé en un ensemble de sous-projets de développements logiciels, de la fondation Eclipse visant à développer un environnement de production de logiciels libre qui soit extensible, universel et polyvalent, en s'appuyant principalement sur Java.

  2. Principale fonctionnalités

    • Plate-formes suportées: Windows, MacOs, Linux
    • Plusieur outils offerts par l'installation de plug-ins
    • Il vise à supporter tout langage de programmation

  3. Forces

    • Il supporte un grand nombre de language de programmation
    • Gratuit - Open Source


  4. Limites

    • On dit qu'il est lent, boggé et instable

NetBeans

  1. Description Gérérale

    • NetBeans est un environnement de développement intégré (EDI), placé en open source par Sun en juin 2000 sous licence CDDL (Common Development and Distribution License)

  2. Principale fonctionnalités

    • Plate-formes suportées: Windows, MacOs, Linux, FreeBSD, Java, Solaris
    • Outils de collaboration par Team Server
    • Il supporte les languages Java, Javasript, PHP, HTML5, CSS, C++, Ruby

  3. Forces

    • Il supporte plusieurs languages de programmation
    • Gratuit - Open Source


  4. Limites

    • Nombre limité de plug-ins
    • Il ne prend pas en charge la modélisation à un niveau élevé
    • Il ne prend pas en charge C#
